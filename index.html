What Is HTML?

HTML stands for Hyper-Text Mark-up Language. It is used to add content to our webpage.

Examples of Contents we can add on HTML includes:

  1. Images
  2. Texts
  3. Videos like:
    • Home-Made Videos
    • You-tube Videos
  4. Audio

What Then is HTML5?

HTML5 is a major revision of the Hyper-Text Mark-up Language (HTML). It serves as the foundation of modern web development. Its purpose is to enhance the structure,functionality, and flexibility of webpages.

HTML5 Contents are as follows:

Sematic Elements:
- These includes elements like <footer>, <article>, <section>, and <aside>. These elements are introduced by HTML5 to provide better structure and clarity to webpage content.

Multi-media Elements:
- This inlude elements like <video>, and <audio>, which allows developers to embed media directly into the webpages without relying on external plugins like flash.
